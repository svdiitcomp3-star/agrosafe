About AgroSafe 🌾

AgroSafe is a smart agriculture protection system designed to help farmers protect crops using modern embedded systems and automation.

🔧 Technologies Used

  • Raspberry Pi Pico
  • Smart Button Interface
  • IoT Concepts
  • Automation & Control Systems

🚀 Features

  • Multiple product versions (V1, V2, V3)
  • Modular design
  • Expandable system
